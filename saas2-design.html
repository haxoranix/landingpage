01 Brand

Calm, structured, trustworthy and efficient

The brand uses soft off-white surfaces, deep green text, lime accents, light borders, grid backgrounds, and rounded UI elements to create a polished B2B SaaS look.

W WorkflowHQ

Logo Usage

Use the lime mark with dark text on light backgrounds.
Use white text with the lime mark on dark green backgrounds.
Do not use busy shadows, gradients, or colored type inside the wordmark.
Do not stretch the mark or change the rounded-square shape.
